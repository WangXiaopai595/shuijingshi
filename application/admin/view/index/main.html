权限系统介绍

# 权限根据《系统配置》-《节点列表》得到

* 节点可禁用，一旦禁用，所有用户都将不能操作该节点功能

* 非开发人员请勿修改节点从属关系


# 管理员权限分配介绍

* 每一个后端管理员都可分配一个角色，该管理员可拥有该角色对应的所有权限

* 管理员角色分配:《管理员列表》-添加/编辑-用户角色


# 为角色分配权限

* 《角色列表》-添加，添加成功后点击列表中的《权限》

* 每一个板块控制器下有多个权限，可根据需要分配不同权限给不同角色

* 注：分配权限时得先给角色分配进入页面的权限，否则拥有权限也操作不了

* 例：给角色A分配管理员编辑权限，那么得先给角色一个进入管理员列表的权限，否则拥有权限也编辑不了对吧!
